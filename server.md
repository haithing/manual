## 前言

本手册采用 Markdown 编制，托管于 Github 。
本手册致力于做到 _覆盖面广_（涉及所有重要的内容），_具体_（给出具体的最常用的例子），以及 _简洁_（避免冗余的内容，或是可以在其他地方轻松查到的细枝末节）。
熟练的掌握本手册所述内容能帮助你节约大量的时间。
